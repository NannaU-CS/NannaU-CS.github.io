## 基本须知

采用匿名/实名发帖的模式，每位评价者从下述维度评价该课程，并自愿附上描述自身情况的 tag。

评课过程中，请尽量考虑到个人能力带来的偏差。尽可能不要轻易给一门课下特别主观的论断（如“这门课很难学”，“所有内容都很简单”）。尽可能使用客观的表述，如“就笔者所知，至少一半同学可以在5h内完成该实验”。

不必拘泥于形式，可以针对自身情况和课程实际调整发帖结构，或增加/略去某些评价内容。

## 不存在的/重复的课程

对于不存在/重复的课程，暂且不要删除相关页面；可在「课程评价」模块中手动注明跳转。例如：

```md
## 课程评价

见[编译原理-冯洋](./冯洋.md)
```

## 评价维度

### 评价者信息

反映评价者选课时前置知识（课程）学习情况、技能水平、自身对该课程的投入程度及预期偏好，可以使用Tag进行标注，有助于读者明确评价和受评价主体，更好地理解后续维度内容。

建议附上最终得分，如不愿详细透露，也可附上或最终得分的大致范围（如60-70, 85-90，90-95）

### 选课信息

同一课程在不同年份不同学期以及不同任课教师的教授下可能大不相同。

- 授课教师的讲课风格影响显然可证，这里不做赘述。
- 存在同一时间不同教师班级的某部分考核内容不一致的情况，如考试占比和考试时间内容各班统一，但平时实验、平时分计算方式、书面作业不同等。
- 据笔者所知，学院开设的某些课程，在秋季学期和春季学期开设时的考核方式和课程侧重可能有相当程度的不一致，推测为面向不同学生群体（本院学生or外院跨选学生）。但也有同一课程在秋季学期和春季学期内容基本一致的情况（如《编译原理》课程）。

为避免歧义，建议在这里注明：

- 选课年份和学期（秋季/春季）
- （如果该课程评价目录下没有对所选班级授课教师做分类）所选班级授课教师
- 可以注明上课时主要讲课教师，以及课堂教学、实验教学上的教师分工情况

### 课程内容

对课程内容的简述。让同学们了解本门课程的**授课内容**，特别是课题、实验、作业会**重点学习考核的知识点**。

### 教师特色

教师的教学风格特色。这一部分偏向于“评教”。

可以包含：

- 教师是否板书、是否读PPT，教师对重要内容的理解程度；
- 对课后问问题、office hour的接受度、态度，所能取得的收获；
- 教师对于陶瓷进组的接受度及建议时间节点，包括教师是否曾经在课程群内发布过招募意向；
- 教师是否可能提供其他帮助，如生涯规划、心理疏导、就业内推等等。

注意，我们希望大家在此处侧重于**客观事实陈述**，尽可能少地加入主观情绪感受，**尤其不提倡人身攻击**。

### 考核与评分标准

理解`工作量`和`学习指南`部分评价信息的重要基础。

通常而言课程考核的分数组成包括以下几类（附各类型组成的评价角度，仅作参考，请撰写评价者自由选用）：

- **出勤**
  - 出勤考察次数
  - 方式：全点名or随机点名or二维码或其他方式签到
  - （如为抽样点名）抽样比例和方式
- **书面作业**
  - 内容：题型，单次题量，难度
  - 次数（或频次）
  - 提交方式：电子or纸质
  - 评分标准
- **实验**
  - 内容简介：是否自选题，是否自选技术栈实现，是否有框架代码
  - 组队要求：是否必须组队，组队人数要求，组队人数是否影响分数计算
  - 次数，各次开始时间及实验时长，前置实验完成度是否影响后续实验
  - 实验下发和提交方式，实验环境配置
  - 提交要求：（开放实验）是否有代码量要求，是否有PPT汇报要求，是否需要助教验收
  - 评分标准：OnlineJudge通过率，功能实现度和验收评分，查重方式，是否有诚信分，是否有按时提交分
- **考试**
  - 考试形式：开卷or闭卷
  - 内容：题型及题型占比，题量，往年卷比例，概念题比例
- **课程论文**
- **MOOC**
- **其他**：如有特殊加分or扣分方式，如上课回答问题发放积分卡片等，也请在此维度分享

以及，请注明每部分组成的分数占比（或注明其为基于总分/某部分分数的额外加减分）。此外：

- *关于考试*：如果对具体内容和比例记忆不清晰，至少可以指出考试时间是否足够（对应是否应该考前刷一刷熟练度）；考试理解性内容偏多还是细节记忆内容偏多（对应备考时是否仅需要理解知识点）
- *关于评分标准*：建议指出作业、考试是否会严格扣除过程分（是否存在预估分数与实际分数相差较大情况）。可以给出多个档次的分数对应标准（如60分、80分、95分）。如果同学们不需求在这一项上尽可能拿分，可以参考更低的标准线。
- *关于课程难度*：详见后文`更具体的评课建议`

### 工作量

对应该课程普遍意义上，在作业、实验、复习等方面需要投入的工时。由于学习效率因人而异，请合理评估大众普遍的工作时长，也可以配合个人能力信息针对自身情况特化描述。

本部分可以配合`评价者信息`部分中关于评价者选课时前置知识和技能水平的信息进行评价。

### 学习指南

对应“如果你已经确定要选修这门课”的情况下，如何尽可能最大化收益或优化投入产出比，可以包含课程网站、推荐攻略、学习方法、考试策略、课程收获等一切于「如何 Survive」相关的话题。

本部分可以配合`评价者信息`部分中关于评价者投入程度和预期偏好的信息进行评价。

总评难度：？

  

**补充：参考难度评价体系：**

1：有点小小难度，我花了一些力气才搞定。/能够被本专业90%同学掌握，如普通概念。/新高考数学正常情况50~120分档/洛谷黄题

3：这太难了，我肯定做不出来。/能够被本专业5%同学掌握，如需要2\~5个小时思考的高档题。/新高考数学正常情况148\~150分档/ICPC区域赛金牌线题/数学高联复赛一等线题/洛谷紫题
