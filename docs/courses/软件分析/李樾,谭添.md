## 评价 1

### 评价者信息

**Tag**：保研；Rank 10%；2025 Spring修课

**Score**：99

### 关于老师

两位 [PASCAL](https://pascal-lab.net) 组的老师都很牛（具体可以读这篇[文章](https://mp.weixin.qq.com/s/8VLCXUtbOV7ZCGC-2zcWAw)了解他们）。两位在非常认真地准备课程内容，课上也有很多 insight。

### 前置知识

基本编程知识（虽然这门课的实验是用 Java 进行实验的，但其实并不需要先会 Java 的知识，笔者是依靠 C++ 基础现学的）。

### 课程内容

课程内容可见[课程网站](https://tai-e.pascal-lab.net/)。此外，[CS自学指南也写了一篇介绍这门课程](https://csdiy.wiki/%E7%BC%96%E7%A8%8B%E8%AF%AD%E8%A8%80%E8%AE%BE%E8%AE%A1%E4%B8%8E%E5%88%86%E6%9E%90/NJU-SoftwareAnalysis/)。

学完此课程，你可以了解数据流分析、指针分析等静态分析的基础内容，对 PL/SE/System 等的研究都大有裨益。

### 作业、考试与得分

作业仅为八次实验，实验见[网站](https://tai-e.pascal-lab.net/intro/overview.html)。

考试则为对平时上课所说的理论知识进行考察，难度不大，且每年开设间题目差距较小。

### 工作量

每次实验约3~6小时即可完成。

### 学习指南

强烈推荐[崔家才](https://static-analysis.cuijiacai.com/)学长的学习笔记。不想听课的同学可以直接读完笔记学完这门课程。
